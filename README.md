# FIRE Kanban

家庭财务看板：用可配置的收支与贷款事件，模拟未来资产负债走势，辅助规划 FIRE 路径并观察异常开销。

所有金额、日期与策略均由你在界面中填写，**代码中不内置任何财务数字**。

## 功能概览

### 参数配置

- **初始理财本金**、**初始活期留存**
- **定存触发器金额**：当活期达到该阈值，且扣除后仍能覆盖下月支出时，自动将等额资金从活期转入理财（流水记为「转账」，不计入收支）
- **预测年数**（1–10 年，滑杆调节）
- **趋势粒度**：月度 / 季度

### 事件列表

左侧维护全部财务事件，点击条目编辑，右上角 **+** 添加新事件。

| 类别 | 说明 |
|------|------|
| 收入 | 定期或单次入账，增加活期 |
| 固定支出 | 从活期或理财扣款（房租、物业等） |
| 贷款 | 本金、年利率、等额本息/等额本金；创建日起计入负债；按还款周期还款 |
| 转账 | 活期 ↔ 理财划转，不影响净资产 |

**日期字段（年 / 月 / 日 + 月初·月末）**

- **首次日期** / **首次还款日**（贷款）
- **末次日期** / **末次还款日**（可勾选「长期」）
- **创建日期**（贷款自该日起计入负债）

**贷款专项**

- **还款周期**（每月 / 每半月 / 每季度 / 每年）结合首末还款日**自动计算期数**
- 可选 **不计入支出**：房贷等仅影响负债与还款，不参与「下月支出」预估（定存触发判断用）

**收入 / 支出** 使用 **发生频率**（每日、每周、每半月、每月、每季度、每年）。

### 视觉看板

- **活期留存**、**理财本金**、**剩余债务**（金额可一键隐藏）
- 副标题显示当前选中的预测时点
- 点击 **趋势图表** 上某一时刻，看板切换为该时点的快照（无悬浮窗）

### 趋势图表

- 理财资产、总负债、累计净资产三条曲线
- 点击图表更新看板；选中点高亮显示

### 流水账

按月份列出每笔事件后的资产快照：

- 绿色：收入  
- 红色：支出 / 还款  
- 白色：转账、贷款计入负债等中性项  

### 设置

- **显示单位**：元 / 万 / 千
- **数据**：导出 / 导入 JSON，或清空参数与事件（显示偏好保留）

## 技术栈

- React 19 + TypeScript + Vite
- Tailwind CSS v4
- Chart.js（趋势图）
- 本地 `localStorage` 持久化（兼容 Safari）

## 快速开始

```bash
npm install
npm run dev
```

浏览器访问终端提示的地址（Vite 已配置 `host: true`，便于 WSL / 局域网访问）。

```bash
npm run build    # 生产构建
npm run preview  # 预览构建结果
```

## 项目结构（简要）

```
src/
  components/     # 参数、事件、看板、图表、流水账、设置
  lib/            # 模拟引擎、贷款摊还、周期 recurrence、格式化
  hooks/          # 持久化状态
  contexts/       # 显示单位
```

核心模拟逻辑见 `src/lib/simulation.ts`：按事件与参数逐月推进活期、理财、负债，并生成快照与流水。

## 数据说明

- 数据仅存于本机浏览器，不会上传服务器
- 存储键：`fire-kanban-state-v1`
- 导入 JSON 会覆盖当前参数与事件，操作前请自行备份

## 设计原则

- Fintech 深色极简 UI（DM Sans）
- 响应式布局，适配桌面与手机
- 金额输入支持多单位展示，内部统一以「元」计算
